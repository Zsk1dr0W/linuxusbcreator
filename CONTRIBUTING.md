# Contributing

Contributions must include tests appropriate to their risk and use a Developer
Certificate of Origin sign-off (`Signed-off-by`) when the project enables its
release workflow. Do not add `Co-authored-by` trailers automatically.

Changes involving block-device writes require threat-model review, failure tests,
and evidence from the virtual test matrix before physical-device testing.

Code copied or adapted from another project must identify its source, revision,
copyright holder, and license in the commit message and source header.
