# Application icon

The initial application icon depicts a USB drive, a boot arrow, and a terminal
prompt. It was generated specifically for Linux USB Creator and then processed
locally to produce a transparent PNG master and freedesktop icon sizes.

Files:

- `io.github.zsk1dr0w.LinuxUsbCreator.png`: 1254 px RGBA master.
- `hicolor/*/apps/io.github.zsk1dr0w.LinuxUsbCreator.png`: generated sizes.

The icon contains no Rufus marks, Linux mascots, text, or third-party logos. It
is distributed under the same GPL-3.0-or-later terms as the project.
